# Overview

We welcome contributions! This guide will help you get started.

## Development Setup

1. Fork and clone the repository
2. Install dependencies: `uv sync --all-extras --group dev`
3. Install pre-commit hooks: `uv run pre-commit install`

[:octicons-arrow-right-24: Development Guide](development.md)

## Testing

Run tests before submitting PRs.

[:octicons-arrow-right-24: Testing Guide](testing.md)

## Guidelines

- Follow the code style (enforced by Ruff)
- Add tests for new features
- Update documentation
- Sign commits (optional but appreciated)
