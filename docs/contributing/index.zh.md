# 贡献指南

欢迎贡献!本指南将帮助你开始。

## 开发设置

1. Fork 并 clone 仓库
2. 安装依赖:`uv sync --all-extras --group dev`
3. 安装 pre-commit hooks:`uv run pre-commit install`

[:octicons-arrow-right-24: 开发指南](development.md)

## 测试

提交 PR 前执行测试。

[:octicons-arrow-right-24: 测试指南](testing.md)

## 指南

- 遵循代码风格(由 Ruff 强制执行)
- 为新功能添加测试
- 更新文档
- 签署 commit(可选但建议)
